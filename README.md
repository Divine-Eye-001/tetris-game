一个基于 JavaScript 和 Canvas 开发的经典俄罗斯方块小游戏（学习练手项目）。

## 项目简介
这是我学习前端开发、Canvas 绘图和微信小程序开发的入门项目，目前已实现俄罗斯方块的核心逻辑。

## 功能特性
- ✅ 经典俄罗斯方块游戏规则
- ✅ 自适应屏幕尺寸适配
- ✅ 支持 T/L/J/O/S/Z/I 7种方块形状
- ✅ 基础分数统计功能

## 技术栈
- **核心语言**：JavaScript (ES6+)
- **绘图技术**：HTML5 Canvas
- **运行环境**：微信小程序
